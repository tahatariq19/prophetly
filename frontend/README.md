# Prophetly Frontend

React-based frontend for Prophetly time series forecasting.

## Stack

- **Framework**: React + Vite
- **Language**: TypeScript
- **Styling**: Tailwind CSS v4 + Shadcn UI
- **State**: Zustand
- **Charts**: Recharts

## Setup

```bash
npm install
```

## Development

```bash
npm run dev
```

## Build

```bash
npm run build
```
